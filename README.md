#  Tableau Analytics Project

A curated collection of Tableau dashboards and worksheets showcasing data modeling, LOD expressions, sets, parameters, joins, and interactive dashboard design using Superstore, Netflix, and other real datasets.

---

##  Highlights
- Analytical tables with combined datasets  
- LOD-based customer order frequency histogram  
- Top 50 customer segmentation using Sets  
- KPI card with YoY Profit Growth & monthly trend  
- Dynamic “View By” parameter (Region/Category/Sub-category)  
- Brand grouping with avg. discount comparison  
- Fully interactive Superstore Dashboard with KPIs, slicers, navigation & dynamic metrics  

---

##  Key Components

### **1. Sales & Profit Matrix**
- Category/Sub-Category vs Region table  
- Sales, Profit, Grand Totals & Percent of Pane Total  
- Sorted by Sales (descending)  

### **2. Netflix Data Join**
- Inner join on Netflix Titles dataset  
- Identified top actor/actress in Sports Movies  
- Listed associated titles  

### **3. LOD Histogram**
- FIXED LOD for customer order frequency  
- Histogram showing customer distribution  

### **4. Top 50 vs Others (Sets)**
- Set based on Top 50 Customers  
- Pie chart comparing Top 50 vs Remaining  

### **5. KPI Card**
- Total Profit (latest year)  
- YoY Profit Growth % with ▲ / ▼ indicators  
- Monthly trend line  

### **6. Parameters & Groups**
- “View By” parameter for dynamic charts  
- Brand groups: Acco, 3M, Samsung, Apple, Xerox + Others  
- Avg. Discount % per group  

### **7. Superstore Dashboard**
- Two-panel layout: logo + filters → visuals  
- Metric parameter (Sales / Profit / # Orders)  
- Year & Region slicers with Show/Hide toggle  
- Category filter action updating all visuals  
- Top 5 products fixed across interactions  
- Trend & Average lines applied
